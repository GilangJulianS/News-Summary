Sudah 12 Jam, Kebakaran di Kings Bandung Belum Juga Padam
http://regional.kompas.com/read/2014/06/24/1246247/Sudah.12.Jam.Kebakaran.di.Kings.Bandung.Belum.Juga.Padam
Selasa, 24 Juni 2014 | 12:46 WIB>>>>>
Kebakaran hebat terjadi di pusat perbelanjaan Kings di Kelurahan Balong Gede, Kecamatan Regol, Bandung, Jawa Barat, Selasa (24/6/2014) dini hari (baca: Pusat Perbelanjaan Kings di Bandung Terbakar). Hingga pukul 12:00 WIB, api belum juga padam.
Titik api diketahui muncul pada Senin kira-kira pukul 23:00 WIB dari sebuah toko kain di pusat perbelanjaan tersebut.
Wadanton III Dinas Kebakaran Kota Bandung Totoy Yuhasmana mengatakan, petugas kesulitan memadam amukan si jago merah karena belum bisa menemukan titik api.
"Kita belum menemukan titik api karena terus terhalang oleh kepulan asap, jadi api sulit dipadamkan," kata Totoy di lokasi kejadian.
Petugas dinas kebakaran Kota Bandung terjun sejak Senin malam. Totoy juga membantah petugas pemadam kebakaran pulang ke markas pemadam pada Selasa dini hari.
"Kita dari malam standby di sini. Kita enggak balik-balik ke markas, justru pukul 02:00 dini hari itu api sedang besar-besarnya. Itu kabar bohong kalau kita ini sempat terhenti melakukan pemadaman, kita terus berjibaku keras melakukan pemadaman sampai sekarang, dari malam api tak padam-padam," tegas Totoy.
Berdasarkan pantauan, petugas pemadam berjibaku memadamkan api, dibantu aparat kepolian dan TNI. Kapolrestabes Bandung Kombes (Pol) Mashudi memprediksi pemadaman api masih akan berlangsung lama.
"Ini api masih besar kayak gini, prediksi padam bisa sampai sore, bisa malam nanti atau besok," katanya.
Akibat terjadinya kebakaran ini, arus lalu lintas terganggu. Beberapa ruas jalan dialihkan. Ribuan warga dan para pengendara menyaksikan pemadaman dari kejauhan.
\\Ringkasan\\
Kebakaran hebat terjadi di pusat perbelanjaan Kings di Kelurahan Balong Gede, Kecamatan Regol, Bandung, Jawa Barat, Selasa (24/6/2014) dini hari (baca: Pusat Perbelanjaan Kings di Bandung Terbakar). 
Titik api diketahui muncul pada Senin kira-kira pukul 23:00 WIB dari sebuah toko kain di pusat perbelanjaan tersebut. 
Berdasarkan informasi yang dihimpun, asap dilaporkan mulai terlihat di bagian lantai I dan kemudian menjalar ke toko-toko tekstil di sekitarnya.
Api, yang diduga berasal dari hubungan arus pedek listrik, baru diketahui setelah membesar pada pukul 00:30. Sebanyak 11 unit mobil pemadam kebakaran dikerahkan untuk memadamkan api.
Polisi nampak kesulitan menghalau masyarakat yang menyemut di sekitar lokasi kejadian. 
Petugas kepolisian memblokir kawasan pertokoan itu, dan menghalau warga yang tetap merangsek ke lokasi kebakaran. 
Akibat terjadinya kebakaran ini, arus lalu lintas terganggu.
Api yang semula membakar bangunan di Jalan Dalem Kaum terlihat menyeberang ke bangunan yang ada di Jalan Kepatihan. 
Bersama anggota TNI dan Polisi yang berjaga, mereka berupaya menyelamatkan barang-barang dagangan yang tersisa. 
Guna memadamkan api, petugas Dinas Pencegahan Kebakaran (Diskar) Kota Bandung dengan dibantu dari pemadam Kota Cimahi dan Kabupaten Bandung menerjunkan 19 mobil pemadam kebakaran.
Berdasarkan pantauan, petugas pemadam berjibaku memadamkan api, dibantu aparat kepolian dan TNI. 
Akhirnya, api berhasil dipadamkan oleh tujuh unit mobil pemadam kebakaran dari Dinas Pemadam Kebakaran Kota Bandung.
