King's Shoping Center Dilalap si Jago Merah
http://bandung.okezone.com/read/2014/06/24/526/1003170/king-s-shoping-center-dilalap-si-jago-merah
Selasa, 24 Juni 2014 03:10 WIB>>>>>
Sebuah mal, King’s Shoping Center, di Jalan Kepatihan, Kelurahan Balong Gede, Kecamatan Regol, dilalap si jago merah, Senin (23/6/2014) tengah malam sekira pukul 23:30 WIB.
Kapolsekta Regol, Kompol Fauzan Sahrir, mengatakan, api pertama kali muncul dari lantai dasar mal yang diisi para pedagang kain dan pakaian.
“Dari keteranga para saksi yang kita dapat, api itu muncul pertama kali dari lantai dasar. Dimana tempat tersebut biasa menjadi lokasi jualan tekstil,” jelas Fauzan saat dikonfirmasi.
Dalam kejadian ini, petugas keamanan yang melihat ada kobaran api sempat melakukan pemadaman dengan alat seadanya. Namun, api terus membesar lantaran lokasi tersebut dipenuhi barang yang mudah terbakar.
Akhirnya, api berhasil dipadamkan oleh tujuh unit mobil pemadam kebakaran dari Dinas Pemadam Kebakaran Kota Bandung.
“Alhamudillah api tidak meluas kelantai ata berkat bantuan keamanan, warga, dan petugas,” katanya.
Saat ini, pihaknya masih mencari tahu asal muasal api yang melahap habis lantai dasar mal tersebut.
“Kerugian kita belum bisa taksir. Kita tunggu kalkulasi dari masing-masing pedagang,” tutupnya.
\\Ringkasan\\
Kebakaran hebat terjadi di pusat perbelanjaan Kings di Kelurahan Balong Gede, Kecamatan Regol, Bandung, Jawa Barat, Selasa (24/6/2014) dini hari (baca: Pusat Perbelanjaan Kings di Bandung Terbakar). 
Titik api diketahui muncul pada Senin kira-kira pukul 23:00 WIB dari sebuah toko kain di pusat perbelanjaan tersebut. 
Berdasarkan informasi yang dihimpun, asap dilaporkan mulai terlihat di bagian lantai I dan kemudian menjalar ke toko-toko tekstil di sekitarnya.
Api, yang diduga berasal dari hubungan arus pedek listrik, baru diketahui setelah membesar pada pukul 00:30. Sebanyak 11 unit mobil pemadam kebakaran dikerahkan untuk memadamkan api.
Polisi nampak kesulitan menghalau masyarakat yang menyemut di sekitar lokasi kejadian. 
Petugas kepolisian memblokir kawasan pertokoan itu, dan menghalau warga yang tetap merangsek ke lokasi kebakaran. 
Akibat terjadinya kebakaran ini, arus lalu lintas terganggu.
Api yang semula membakar bangunan di Jalan Dalem Kaum terlihat menyeberang ke bangunan yang ada di Jalan Kepatihan. 
Bersama anggota TNI dan Polisi yang berjaga, mereka berupaya menyelamatkan barang-barang dagangan yang tersisa. 
Guna memadamkan api, petugas Dinas Pencegahan Kebakaran (Diskar) Kota Bandung dengan dibantu dari pemadam Kota Cimahi dan Kabupaten Bandung menerjunkan 19 mobil pemadam kebakaran.
Berdasarkan pantauan, petugas pemadam berjibaku memadamkan api, dibantu aparat kepolian dan TNI. 
Akhirnya, api berhasil dipadamkan oleh tujuh unit mobil pemadam kebakaran dari Dinas Pemadam Kebakaran Kota Bandung.
